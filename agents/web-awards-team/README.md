# Web Awards Team

Grupo de agentes especializado en crear o adaptar webs para volverlas mucho mas atractivas, memorables y cercanas a un nivel "award-worthy".

Este equipo esta pensado primero para este repo (`Astro + Tailwind 4 + Preact`), pero la mayoria de roles son reutilizables en otros proyectos.

## Objetivo

No producir "efectos bonitos" aislados, sino experiencias con:

- direccion artistica clara,
- jerarquia visual fuerte,
- motion con sentido,
- superficies refinadas,
- una base tecnica solida.

## Reglas comunes

- No aceptar direcciones visuales genericas.
- No usar motion porque si; cada gesto debe mejorar orientacion, tono o feedback.
- No introducir 3D o canvas si el retorno narrativo es bajo.
- Mantener fallback para `prefers-reduced-motion`.
- En este stack, priorizar Astro estatico y usar Preact solo cuando la interaccion lo justifique.
- Aprovechar `astro:transitions` y `astro:assets` antes de añadir complejidad innecesaria.

## Como pedirmelo despues

Ejemplos utiles:

- `Usa el Web Awards Team para auditar la home y proponer una direccion visual seria.`
- `Activa el equipo para transformar /partidos en una pagina mucho mas editorial sin perder claridad.`
- `Quiero modo flagship para la home: empieza por visual direction y termina con integracion Astro.`
- `Usa editorial-voice-director para que los textos suenen mas periodisticos y menos roboticos.`
